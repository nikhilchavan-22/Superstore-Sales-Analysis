# Superstore Sales Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Superstore dataset using Python to identify sales patterns, profit trends, and customer insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
• Sales analysis by category and sub-category  
• Profit analysis across regions  
• Customer segment insights  
• Trend analysis for sales and profit
